# Unit 1 – Introduction to Secure Systems Architecture

## Overview

In Unit 1, the module introduced the concept of secure systems architecture from the perspective of operating systems and distributed systems. The lecture explained how modern computing systems have evolved to support new applications and user requirements, and why security must be considered as part of system design.

The unit also introduced the difference between operating systems and distributed systems, including how processes, threads, and schedulers work. In addition, we learned about containers and how tools such as Docker can be used to isolate applications and improve system security.

---

## Seminar Activity

During the seminar, we were assigned an operating system to research.  
The task required us to investigate:

- Who created the operating system
- When it was created
- What its purpose was
- Its unique features
- Whether it is still used today

I prepared a short presentation explaining the history and characteristics of the selected operating system. This activity helped me understand how operating systems have evolved and why security features became more important over time.
